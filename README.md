## 🏛 Философия проекта

### **«Степной Примитивизм с Интеллектом Аксакала»**

Мы отказались от «жирных» облачных вычислений в пользу **Edge AI** — 
интеллект работает прямо на периферийных устройствах, без зависимости 
от нестабильных каналов связи.

**Принципы:**
- 🌐 **Децентрализация** — рой агентов координируется без центрального сервера
- 🔒 **Суверенитет данных** — Federated Learning без передачи сырых данных
- 🎯 **Explainability** — каждое решение объясняется оператору
- ⚡ **Быстродействие** — решение за «семь вдохов» (миллисекунды)

---

## 🏗 Архитектура


### Четыре этапа принятия решений:

1. **Triage** — классификация и приоритизация входящих задач
2. **Bidding** — аукционное распределение между автономными агентами
3. **Consensus** — консенсусная валидация решений роя
4. **Explainability** — формирование логического обоснования
