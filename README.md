🐜 QumyrsqaCore: Sovereign Swarm Intelligence (ИС УТО)Интеллектуальная система управления логистикой спецтехники на месторождениях на базе роевого интеллекта и Edge AI.📌 О проектеQumyrsqaCore — это децентрализованная когнитивная экосистема управления флотом спецтехники в условиях «Майнинга 4.0». Система решает классическую задачу маршрутизации (VRP) через использование автономных агентов, принимающих решения на «краю» (Edge), даже при полном отсутствии связи.🛡️ Юридический щит и СуверенитетАрхитектура и логика системы официально зарегистрированы:Свидетельство: № 68948 от 19.03.2026 (НИИС РК).Автор: Жумагулов Нурлан Бакытович.Концепция: Суверенный Роевой Интеллект (Sovereign Swarm Intelligence).🚀 Ключевые инновации (Killer Features)Edge AI & Zero-CAPEX: Использование вычислительной мощности обычных смартфонов водителей (через Termux) вместо закупки дорогих промышленных контроллеров.Алгоритм Бейсекенова: Многокритериальная оптимизация выбора техники, учитывающая не только расстояние, но и приоритет, износ и надежность:$$Score = (W_{time} \cdot ETA) + (W_{dist} \cdot Dist) + (W_{pri} \cdot Priority)$$Протокол «Ізеу» (Izeu): Цифровой консенсус роя. Машины сами «договариваются», кто берет задачу, основываясь на данных о местоположении и готовности.Explainability AI (XAI): Модуль объяснимости переводит сухие расчеты в текст для диспетчера: «Выбран Самосвал-05: экономия 12 км пробега, водитель отдохнул».🏗️ Архитектура системыПроект построен по модульному принципу:Backend: FastAPI (Python 3.12+) — обеспечивает REST API для интеграции.Spatial Engine: Scipy KD-Tree для мгновенной привязки GPS-координат к дорожному графу.Swarm Engine: NetworkX для расчета маршрутов и динамической стигмергии (цифровые феромоны).Frontend: Streamlit 3D Dashboard для визуализации роя в реальном времени.🎬 Сценарии демонстрации (По ТЗ хакатона)Система успешно отрабатывает три критических сценария:Сценарий 1 (Срочность): Заявка High Priority -> Рой игнорирует экономию и присылает ближайшую свободную машину.Сценарий 2 (Оптимизация): Сравнение Baseline (просто ближайший по прямой) vs QumyrsqaCore (оптимальный путь по графу).Сценарий 3 (Многозадачность): Объединение 3 заявок в одном районе в один маршрут для одной машины.🛠️ Установка и запускBash# Клонируйте репозиторий
